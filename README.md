Graph-based Visual Programming core module for workers

## How to initialize boilerplate

```shell
./init.sh
```

## Developing

```shell
## Create a local Python virtual environment.
./python

## Black formatting
./black.sh

## PEP8 linting
./flake8.sh

## Sort import order
./isort.sh

## Type checking
./mypy.sh

## Unit testing
./pytest.sh
```
